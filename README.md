# Python LMS Management System

## Overview
This is a **Learning Management System (LMS)** built using Python. It allows educational institutions to manage courses, instructors, and students efficiently. Users can track learning progress, enroll in courses, and manage content easily.

## Features
- User roles: Admin, Instructor, Student
- Course management (Add, Update, Delete)
- Student enrollment and progress tracking
- Instructor assignment to courses
- Simple dashboard for Admin and Instructors
- CLI (Command Line Interface) or Web interface (depending on setup)
